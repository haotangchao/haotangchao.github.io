# C语言学习笔记

1. `int main(void){}`是现在编译器认可的主函数定义格式

2. 八进制是加0，十六进制是加0x，如十进制数100表示成八进制是0144,表示成16进制是0x64

3. 在输出整型时，要在八进制和十六进制值前显示0和0x的前缀，要在转换说明中加入#，比如`printf("%d,%#o,%#x",num,num,num)` 为100,0144,0x64

4. 整数类型有：
   * int
   * short int(short) 占位符%hd
   * long int(long) 占位符%ld(d 可以改成x 和 o 分别表示十六进制和八进制)
   * long long int (long long)
   * unsigned int (unsigned) 占位符%u
   * unsigned long、unsigned short、unsigned long long
   * 以上1、2、3、4条均可在前面加signed以强调有符号

5. 浮点数类型有：
   * float、double 占位符(%f、%e、%a)
   * long double 占位符(%Lf、%Le、%La)
   * 用%e打印指数记数法的浮点数，例如3.68E5
   * 用%a打印十六进制的指数记数法的浮点数

    * 关闭位：~& MASK中为1的位在结果中都被清空为0

    * 切换位：^ MASK中为1的对应位都被切换了，MASK为0的位相对应的位不变。

    * 检查位的值：`if((flags & MASK) == MASK)`

      ​				`puts("Wow");`

    * 左移 << （左边的二进制位丢弃，右边的补0）

    * 右移 >> （右边的二进制位丢弃，左边的补0）

33. 声明和定义的区别：

    * 声明：描述对象的类型，用于指代其他地方定义的对象。声明相当于普通的声明，它所说明的并非自身。
    * 定义：确定对象的类型并分配内存，用于创建新的对象。定义相当于特殊的声明，它为对象分配内存。

35. 有这么两个语句：

    `int a[4][5];`

    `int** p;`

    `p = a;		//错！！！！！不能这么用`

    为什么？因为**类型不同！**a是数组的数组，p是指向指针的指针，

    那若是要使用p作为数组该怎么用？用malloc

    `p = (int **)malloc(sizeof(int*)*5);`

    `for(int i = 0; i < 4; i++){`

    `	p[i] = (int *)malloc(sizeof(int) * 4);`

    `}`

    之后就可以使用`p[i][j]`了

    若是非要令指针指向一个数组，那得这么声明p：

    `int (*p)[5];`

    即一个指向大小为5的数组的指针，注意`[ ]`的优先级高于`*`，所以与`int* p[5]`区分！，这是一个大小是5的用来存放指针的数组。

36. 可以利用`memcpy()`函数进行二维数组的整行交换。例子：

    `int matrix[m][n];`

    `int * t;`

    `t = (int *)malloc(sizeof(int) * n);`

    `for(int i = 0; i < m / 2; i++){`

    ​	`	memcpy(t, matrix[i], sizeof(int) * n);`

    ​	`memcpy(matrix[i],matrix[n - i - 1], sizeof(int) * n);`

    ​	`memcpy(matrix[n - i - 1], t, sizeof(int) * n);`

    `}`

    `free(t);`
